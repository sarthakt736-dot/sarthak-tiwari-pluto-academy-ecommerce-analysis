# 🛒 E-Commerce Sales Analysis
**Pluto Academy | Data Analytics Internship — Project 01**

**Developed by:** Sarthak Tiwari

## Problem Statement
Analysis of 12 months of Olist Brazilian E-Commerce data to identify 
revenue trends, top categories, regional performance, and customer satisfaction.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI (Dashboard)
- Google Colab

## Dataset
Olist Brazilian E-Commerce Dataset — Kaggle

## Key Findings
- Top Category: Health & Beauty (BRL 14.1L — 9.2% of revenue)
- Peak Month: November 2017 (BRL 11.5L)
- Top Region: São Paulo — 37.4% of total revenue
- Average Order Value: BRL 159.83
- Average Review Score: 4.08 / 5.0
